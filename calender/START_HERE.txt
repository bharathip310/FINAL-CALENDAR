üìù DATABASE SETUP EXAMPLES
==========================

LOCAL MONGODB:
--------------
1. Install from: https://www.mongodb.com/try/download/community
2. Run: mongod
3. In .env, use: MONGODB_URI=mongodb://localhost:27017/calendar_db
4. Run: npm start

MONGODB ATLAS (RECOMMENDED FOR BEGINNERS):
------------------------------------------
1. Go to: https://www.mongodb.com/cloud/atlas
2. Sign up with Google/GitHub/email (free)
3. Create M0 cluster (always free)
4. Click "Connect" > "Connect your application"
5. Copy connection string
6. Replace <username> and <password>
7. Paste in .env MONGODB_URI
8. Run: npm start

Example Atlas URL:
MONGODB_URI=mongodb+srv://user:pass@cluster.mongodb.net/calendar_db
