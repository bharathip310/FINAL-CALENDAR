üîß INSTALL MONGODB (if not done)
===============================

Windows:
1. Download: https://www.mongodb.com/try/download/community
2. Run installer and follow prompts
3. Install as service (checked by default)
4. Run: mongod
5. Should say "Waiting for connections on port 27017"

Mac:
1. Download: https://www.mongodb.com/try/download/community
2. Open .tgz file
3. Follow brew installation or manual steps
4. Run: mongod

Linux (Ubuntu):
$ sudo apt-get install mongod
$ sudo systemctl start mongod

MongoDB Atlas (Cloud - Easiest):
1. Go to: https://www.mongodb.com/cloud/atlas
2. Sign up (free tier available)
3. Create cluster M0 (free forever)
4. Copy connection string
5. Paste in .env MONGODB_URI

üêõ DEBUGGING TIPS
================

If server won't start:

‚ùå Error: "Cannot connect to MongoDB"
$ Check: Is MongoDB running?
$ Check: Is MONGODB_URI correct in .env?
$ Fix: mongod --version (verify installed)

‚ùå Error: "EADDRINUSE :::3000"
$ Another app using port 3000
$ Fix: Change PORT=3001 in .env
$ Or kill: netstat -ano | findstr :3000

‚ùå Error: "MONGODB_URI not found"
$ Check: .env file exists
$ Fix: Create .env with correct connection string

‚ùå Error: "Authentication failed"
‚úì Check MongoDB Atlas credentials
‚úì Check IP whitelist (allow 0.0.0.0/0)
‚úì Check database user permissions

üéØ QUICK EXAMPLES
=================

Create user in MongoDB:
db.users.insertOne({
  username: "newuser",
  password: "hashedpassword",
  email: "new@example.com",
  name: "New User",
  role: "student",
  createdAt: new Date()
})

Find all admins:
db.users.find({ role: "admin" })

Count total users:
db.users.countDocuments()

Update a user:
db.users.updateOne(
  { username: "admin" },
  { $set: { email: "newemail@example.com" } }
)

Delete a user:
db.users.deleteOne({ username: "student1" })
