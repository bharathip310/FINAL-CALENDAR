BACKEND (server.js)
-------------------
New API Endpoints:

1. GET /api/chat/history
   Purpose: Retrieve chat history for user
   Auth: Optional (works for both logged-in and guest users)
   Response: { success: true, messages: [...] }

2. POST /api/chat/message
   Purpose: Send a message and get bot response
   Auth: Optional
   Body: { message: "user message" }
   Response: { 
       success: true, 
       userMessage: "...", 
       botResponse: "..." 
   }

3. POST /api/chat/clear
   Purpose: Clear chat history
   Auth: Optional
   Response: { success: true, message: "Chat history cleared" }
